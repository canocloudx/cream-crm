Option 1: Matched Background Color

Recommended
How it works: The logo's black background is replaced with the exact sidebar color (#0a0a0c), making it blend seamlessly.

Pros: Perfect integration, no CSS tricks needed, works everywhere.
Cons: If sidebar color changes, logo needs to be regenerated.

Option 2: CSS Blend Mode

Alternative
How it works: Uses CSS mix-blend-mode: screen to make black transparent while keeping the neon glow.

Pros: Adapts to any background automatically.
Cons: Glow effect may appear slightly different, browser compatibility considerations.
